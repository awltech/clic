CLiC Command Line Client - Eclipse Framework

What?

CLiC (Command Line Client) is an Eclipse framework allowing to deal with command lines.

How?

CLiC is only a generic framework, so it provides an easy way to plug new commands, and we're counting on you to enrich the framework with your contributions!

Who?

CLiC is maintained by a few Worldline developers, and needs you to be alive! So please let us know if you'd like to contribute something so we can promote here right here!
